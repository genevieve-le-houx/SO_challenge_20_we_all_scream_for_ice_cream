# Stackoverflow challenge #20: We all scream for Ice Cream! 

This is my submission to Stackoverflow challenge #20: We all scream for Ice Cream! 

https://stackoverflow.com/beta/challenges/79962428/challenge-20-we-all-scream-for-ice-cream

The goal of the challenge is to compute how many tubs of ice cream of each flavor are needed given a list of guests and their flavor choice.

## Execution

The code is done using Python 3.14.3. To execute, simply run main.py. No additional libraries needed
